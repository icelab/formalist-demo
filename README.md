# Formalist Demo

### First-time setup

```
./bin/setup
```

### Running the app

```
bundle exec shotgun -p 3000 -o 0.0.0.0 config.ru
npm start
```

Then load [http://localhost:3000/demo](http://localhost:3000/demo).
